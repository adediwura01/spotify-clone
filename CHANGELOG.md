# Changelog

All notable changes made during the development of this project are documented here.

---

## Version 0.1.0 — Initial Foundation

### Project Setup

* Initialized React project with Vite.
* Organized scalable folder structure.
* Configured React Router.
* Added reusable layout components.

### Sidebar

* Built responsive sidebar navigation.
* Added Library section.
* Added Liked Songs navigation.
* Customized sidebar scrollbar.

### Home Page

* Created reusable Hero component.
* Built reusable playlist sections.
* Added reusable playlist cards.
* Integrated local playlist images.

### Player

* Built reusable bottom player.
* Added Song Info component.
* Added Playback Controls.
* Added Extra Controls.
* Introduced global Player Context.
* Clicking a playlist now updates the active player.

### UI Improvements

* Improved search bar styling.
* Customized main content scrollbar.
* Added reusable animation architecture.
* Added Framer Motion page entrance animations.

### Architecture
- Renamed Hero component to Featured to support a scalable featured carousel architecture.
